# Support

Use GitHub Issues for support related to this repository.

## Open an Issue For

- Broken site behavior
- Incorrect generated data
- Missing or malformed source definitions
- Build, sync, derive, or verify failures
- Requests to track a new source or harness

## Before Opening an Issue

- Check the README and contribution docs
- Confirm the problem on the latest `main`
- Include reproduction steps when reporting a bug
- Include commands run and relevant logs if the issue is build-related

## Where To Go

- Bug reports: [GitHub Issues](https://github.com/jianxiaoyitech/agent-harness-extensions/issues)
- Source requests: [GitHub Issues](https://github.com/jianxiaoyitech/agent-harness-extensions/issues)
- Contributions: [`CONTRIBUTING.md`](./CONTRIBUTING.md)
